== Description ==

This plugin protects registration, login and comment forms from spambots by adding a simple math question to it ("what is 4+3 ?"). This approach should give you 100% anti-spam protection to your sites.

== Installation ==

1. Upload `block-spam-by-math` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy :)
